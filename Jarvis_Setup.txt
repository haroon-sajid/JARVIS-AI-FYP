```
1. Install Python 3.10.10 in the terminal of VS Code:
   ```
   pip install python 3.10.10
   ```

2. Create a folder named `J.A.R.V.I.S_A.I` in the C: directory:
   ```
   mkdir C:\J.A.R.V.I.S_A.I
   ```

3. Paste all 5 folders in this folder.

4. Install all libraries:
   ```
   pip install -r requirements.txt
   ```

5. Go to the MAIN folder and run the `Main.pyw` file.

6. When Jarvis speaks the greeting, you say the command 'Wakeup'.

7. After that, Jarvis will start the conversation.
```
